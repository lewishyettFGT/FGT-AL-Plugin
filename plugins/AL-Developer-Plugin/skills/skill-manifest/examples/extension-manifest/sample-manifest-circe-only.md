### Custom Tools (this extension)

`cltLeads` (Page 50100): CRUD operations on leads plus the `ConvertToCustomer` bound action that triggers Codeunit 50100 to convert a qualified lead into a customer record.

`cltLeadActivities` (Page 50101): CRUD operations on activity log entries linked to leads.

`cltLeadPipeline` (Query 50102): Read-only pipeline aggregation, useful for agent responses that summarize deal flow.

### Standard BC Tools (relevant to this domain)

`customers` (standard): The agent will reference customer records when a lead is converted or when validating duplicate contacts.

`salesOrders`, `salesQuotes` (standard): Post-conversion, the agent may create quotes or orders on behalf of the salesperson.

`contacts` (standard): Leads reference contacts via the Contact No. field; the agent needs this for contact lookup and enrichment.
